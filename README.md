# ShopSwift-Full-Stack-E-Commerce-App
A complete MERN + TypeScript ecommerce application with authentication, product management, cart system, orders, and admin features.

This project includes:

1)Backend: Node.js, Express, MongoDB, JWT Auth, TypeScript

2)Frontend: React, Zustand, REST API integration


**Features**: Login, Register, Products, Cart, Orders, Admin Panel

**Tech Stack**
**Frontend**
1)React.js

2)Zustand (State Management)

3)Axios

4)React Router

5)Custom Components

Backend
1)Node.js

2)Express.js

3)MongoDB + Mongoose

4)JWT Authentication
5)Authentication Features
6)Register new users

7)Login with JWT

8)Protected routes

9)Admin-only routes

10)Token-based session handling
11)TypeScript

12)Middlewares (Auth, Admin, Error Handling)

**E‑Commerce Features**
1)Product listing

2)Product details

3)Add to cart

4)Remove from cart

5)Update cart quantity

6)Checkout flow

7)Order creation

**API Endpoints (Short Overview)**
Auth
1)POST /api/auth/register

2)POST /api/auth/login

**Products**
1)GET /api/products

2)POST /api/products (Admin)

**Cart**
1)GET /api/cart

2)POST /api/cart/add

3)DELETE /api/cart/remove/:id

**Orders**
1)POST /api/orders

2)GET /api/orders/my-orders

**Testing Tools**
Thunder Client / Postman
